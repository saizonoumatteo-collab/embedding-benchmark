VS
🚀 Benchmark de vitesse — Mesure le temps de génération d'un embedding. Latence moy. = temps par texte. P95 = percentile 95 (stabilité). Throughput = textes/seconde. Les box plots montrent la distribution complète des latences.
🎯 Semantic Textual Similarity — Corrélation de Pearson r entre la similarité cosinus prédite et des scores annotés humainement (30 paires). r > 0.7 = excellent · 0.5–0.7 = bon · < 0.5 = faible.
🔎 Information Retrieval — Pour chaque requête, les documents sont classés par similarité. NDCG@5 récompense les documents pertinents bien classés. Recall@5 = fraction récupérée dans le top-5. MRR = rang du 1er document pertinent. Scores > 0.8 = excellent.
📦 Classification par embeddings — Teste si les embeddings de même catégorie se regroupent. Précision centroïde = affectation au vecteur moyen le plus proche. Silhouette = séparation globale des clusters. La matrice de confusion montre les erreurs de classification.
🛡 Robustesse aux paraphrases — Similarité intra-groupe (paraphrases) vs inter-groupe (phrases différentes). Ratio de discrimination = intra/inter. > 2.0 = excellent · > 1.5 = bon.
🌍 Alignement multilingue — Similarité cosinus entre traductions EN↔FR/ES/DE/IT/PT. Score d'alignement = sim_traduction − sim_non-traduction. > 0.3 = bon modèle multilingue · ≈ 0 = monolingue.
🔄 Conscience de la négation — Teste si le modèle distingue les affirmations opposées ("j'adore" vs "je déteste"). La plupart des modèles capturent le sujet mais pas la polarité. Similarité négation proche de 1 = le modèle ignore l'opposition. Score de conscience = 1 − sim_moy.
📉 Topic Drift — Mesure si la similarité décroît de façon monotone quand le sujet s'éloigne progressivement de l'ancre. Un bon modèle produit un gradient sémantique régulier. Score de monotonicité = fraction des paires consécutives où sim diminue.
📍 Visualisation des embeddings — Réduction dimensionnelle 2D. PCA (rapide, linéaire) vs t-SNE (lent mais révèle mieux les clusters non-linéaires). Les points de même couleur = même catégorie. Passez la souris pour voir le texte.
